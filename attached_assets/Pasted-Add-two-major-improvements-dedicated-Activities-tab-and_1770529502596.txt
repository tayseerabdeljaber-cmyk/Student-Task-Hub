AI GENERATION ALGORITHM (Enhanced):

INPUT:
1. All assignments from backend with due dates and estimated effort
2. All classes from Activities (locked blocks)
3. All recurring activities from Activities (some locked, some flexible)
4. Sleep schedule (locked block)
5. User preferences:
   - Study time preference (morning/afternoon/evening/night)
   - Buffer time settings
   - Break frequency
   - Spread vs. cram tendency
6. Current date/time
7. Generation timeframe (today, week, 2 weeks, month)

PROCESSING:

STEP 1 - Create time grid:
- Generate hourly grid for timeframe
- Mark all locked blocks (classes, locked activities, sleep)
- Mark all flexible activity blocks (can be moved if needed)
- Identify all free time slots

STEP 2 - Estimate effort for each assignment:
```typescript
function estimateEffort(assignment) {
  const baseEffort = {
    'homework': 2,      // hours
    'reading': 1,
    'quiz_prep': 1.5,
    'exam_prep': 6,     // spread over multiple days
    'lab': 2.5,
    'project': 8,       // spread over multiple days
    'essay': 4,
  };
  
  // Adjust based on course difficulty (from user ratings or defaults)
  const courseDifficulty = getCourseRating(assignment.courseId);
  return baseEffort[assignment.type] * courseDifficulty;
}
```

STEP 3 - Prioritize assignments:
```typescript
function priorityScore(assignment) {
  const daysUntilDue = (assignment.dueDate - now) / (24 * 60 * 60 * 1000);
  const effort = estimateEffort(assignment);
  
  // Higher score = more urgent
  return (effort / daysUntilDue) * assignment.weight;
}
```

Sort assignments by priority score descending

STEP 4 - Schedule each assignment:

For each assignment (highest priority first):
1. Determine when to start:
   - Exams: Start 3-5 days before
   - Projects: Start 5-7 days before  
   - Homework: Start 2-3 days before
   - Reading: Start 1-2 days before
   
2. Determine session count:
   - Break long assignments (4+ hours) into multiple sessions
   - Max 2 hours per session (with breaks)
   - Min 30 minutes per session
   - Example: 6 hour exam prep = 3 sessions of 2 hours each
   
3. Find best time slots:
   - Filter slots by user's preferred study times
   - Prefer slots where user is most productive (based on past completion)
   - Avoid back-to-back study sessions (add breaks)
   - Spread sessions across different days when possible
   - Don't schedule after 10 PM unless user is night owl
   
4. Place study blocks:
   - Place in optimal time slots
   - If no optimal slots available, use secondary preferences
   - If still can't fit, show warning about overloaded schedule
   
5. Add study breaks:
   - Every 2 hours of continuous study: Add 10-15 min break
   - After 4 hours total study in a day: Add 30 min longer break

STEP 5 - Balance the schedule:
- Check daily load for each day
- If any day exceeds 8 hours of study + classes + activities:
  * Mark as "overloaded"
  * Suggest moving flexible activities
  * Offer to spread work to adjacent days
- Try to keep weekends lighter (unless user opts in)
- Ensure at least 1-2 hours of free time each day

STEP 6 - Optimize:
- Look for inefficiencies:
  * Study blocks with large gaps between them (consolidate)
  * Same-course study sessions too close together (space out)
  * Study right before that class (good) vs. random times (less good)
- Apply optimizations

OUTPUT:
- New schedule blocks added to calendar
- Warnings/suggestions for overloaded days
- Summary statistics

// Activity (class, recurring commitment, one-time event)
interface Activity {
  id: string;
  name: string;
  type: 'class' | 'study' | 'work' | 'exercise' | 'meal' | 'sleep' | 'social' | 'personal';
  icon: string; // emoji
  color: string;
  
  // Scheduling
  recurring: {
    frequency: 'once' | 'daily' | 'weekly' | 'custom';
    daysOfWeek?: string[]; // ['Monday', 'Wednesday', 'Friday']
    startDate: Date;
    endDate?: Date; // optional, for semester-long classes
    customPattern?: string; // for complex patterns
  };
  
  timeSlot: {
    startTime: string; // "09:00"
    endTime?: string;  // "09:50"
    duration?: number; // minutes, alternative to endTime
    flexible: boolean; // can AI move this?
  };
  
  // Metadata
  location?: string;
  priority: 'low' | 'medium' | 'high' | 'critical';
  bufferBefore: number; // minutes
  bufferAfter: number;  // minutes
  
  // Notifications
  reminders: {
    enabled: boolean;
    minutesBefore: number;
  }[];
  
  // Tracking
  completions: {
    date: Date;
    completed: boolean;
    actualDuration?: number; // for learning
  }[];
}

// Generated Schedule Block (instance of activity on specific date)
interface ScheduleBlock {
  id: string;
  activityId: string; // reference to Activity
  assignmentId?: string; // if it's a study block for assignment
  
  date: Date;
  startTime: Date;
  endTime: Date;
  
  isGenerated: boolean; // true if AI generated, false if user created
  isLocked: boolean;    // can't be moved by AI regeneration
  isCompleted: boolean;
  
  // For study blocks
  metadata?: {
    assignment: string;
    estimatedTimeRemaining: number;
    actualTimeSpent: number;
    difficulty: 'easy' | 'medium' | 'hard';
  };
}

// User Preferences
interface SchedulePreferences {
  // Time preferences
  productiveHours: {
    morning: boolean;    // 6 AM - 12 PM

  };
  
  // Sleep
  sleepSchedule: {
    bedtime: string;  // "22:00"

    hoursNeeded: number; // 8
  };
  
  // Study habits
  studySession: {
    maxDuration: number; // minutes, default 120
    minDuration: number; // minutes, default 30
    breakFrequency: number; // minutes, default 120 (break every 2 hrs)
    breakDuration: number; // minutes, default 15
  };
  
  // Scheduling behavior
  scheduling: {
    startAssignmentsEarly: boolean; // vs. closer to deadline
    spreadExamPrep: boolean; // vs. cram
    allowWeekendStudy: boolean;
    defaultBufferTime: number; // minutes
  };
  
  // Learned patterns
  learned: {
    actualCompletionRate: number; // % of scheduled blocks actually completed
    averageStudySpeed: number; // multiplier for estimates (1.0 = on pace, 1.5 = faster, 0.7 = slower)
    bestStudyTimes: string[]; // ["09:00", "14:00"] - times with highest completion rate
  };
}
